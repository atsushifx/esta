### 🗺️ 読み方のガイドライン

1. 開発マシンのセットアップが終わっていない場合は、上から順に読み進めてください。
2. セットアップ済みであれば、必要な章のみを参照してください。
3. 質問・改善提案は issue や PR にて歓迎します。
